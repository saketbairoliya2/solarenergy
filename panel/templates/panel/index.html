This application have got demo data for unit with IDs: 4, 5, 6

API for accessing Performance data for individual system follows this style:

https://solarenergy.herokuapp.com/panel/{unit_id}?date={dd-mm-yyyy}

API for sending mails for a given unit follows this style:

https://solarenergy.herokuapp.com/panel/{unit_id}/mail?date={dd-mm-yyyy}

Note: Acceptable date format of making request is DD-MM-YYYY

Further a mail server is configured to send mail at 8 PM to recipents of mail(Both of the paramaaters can be cahnged in code as and when needed)

Database as of now is populated with data from date 20-03-2107 till 20-04-2017 for testing
